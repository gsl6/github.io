JDBC:

官方定义的一套操作所有关系型数据库的规则（接口）；各个数据库产商去实现这个接口，提供数据库驱动

jar包，我们可以使用这套接口编程，真正执行的代码是驱动jar包中的实现类

JDBC的操作做步骤

  1. 新建一个名为lib的文件夹，用于存放jar包

  2. 下载驱动包：https://dev.mysql.com/downloads/connector/j/5.1.html

  3. 添加项目依赖：选中lib目录下的jar包右击->Add as library->设置添加的驱动名字

  4. 验证添加依赖是否成功：file->Project Structure->Libraries,选中之后看右边是否出现刚才添加的jar包

JDBC开发流程

  1. 通过反射来加载驱动包

  2. 使用DeriverManger来获取连接对象

  3. 获取可以执行的SQL语句的对象statement

  4. 执行SQL语句

  5. 得到返回的结果

  6. 释放资源

          
      import com.mysql.jdbc.Driver;
      import java.sql.*;
      public class test {
          public static void main(String[] args) {
              Connection con=null;
              Statement sta=null;
              ResultSet re=null;
              try {
                  Class.forName("com.mysql.jdbc.Driver");  //通过反射来加载驱动
                  //获取与数据库的连接对象con
                  con=DriverManager.getConnection("jdbc:mysql://localhost:3306/student?characterEncoding=utf-8","root","123");
                  sta=con.createStatement();//通过连接对象得到SQL的执行对象sta
                   re=sta.executeQuery("selet *from user");//执行SQL语句，得到执行结果集
                  while(re.next()){                           //遍历结果集得到数据
                      int uid=re.getInt("uid");
                      String uname=re.getString("uname");
                      String upwd=re.getString("upwd");
                      System.out.println(uid+"\t"+uname+"\t"+upwd);
                  }
              } catch (ClassNotFoundException e) {
                  e.printStackTrace();
              } catch (SQLException e) {
                  e.printStackTrace();
              }finally {    //关闭资源，先调用的先关闭，关闭前判断对象是否存在
                  try {
                      if(re !=null){
                          con.close();
                      }
                      if(sta !=null){
                          sta.close();
                      }
                      if(con !=null){
                          re.close();
                      }
                  } catch (SQLException e) {
                      e.printStackTrace();
                  }
              }
          }
      }

Jdbc中用到的对象

1.DriverManager对象：驱动管理对象

功能;

  • 注册驱动：告诉程序使用哪一个数据库驱动jar; mysql5之后的驱动jar包可以省略注册驱动的步骤

  • 获取数据库连接

2.Connection对象
作用：完成客服端与数据库的交互

常用的方法：

​ 1).获取执行sql对象

  • createStatement() //创建向数据库发送sql的statement对象

  • prepareStatement(sql) //创建向数据库发送预编译sql的PrepareSatment对象

  • prepareCall(sql) //创建执行存储过程的callableStatement对象

    2).管理事务：

  • setAutoCommit(boolean autoCommit) //设置事务自动提交

  • commit() //提交事务

  • rollback() //回滚事务

    ​

3.Statement对象

作用：用于向数据库发送sql语句，对数据库的增删改查都可以通过此对象发送sql语句完成

常用的方法:

  • executeQuery(String sql) //执行DQL(SELECT)语句

  • executeUpdate(String sql) //执行DML语句（增删改），返回值是影响的行数(可以判断DML语句是否执行成功)

  • addBatch(String sql) //把多条sql语句放进同一个批处理中

  • executeBatch() //向数据库发送一批sql语句执行

4.ResultSet对象

作用：ResultSet对象代表sql语句的执行结果，当Statement执行executeQuery()时会返回一个ResultSet对象

ResultSet对象维护了⼀个数据⾏的游标，调⽤ResultSet.next()⽅法可以让游标指向具体的数据⾏，进⾏获取该⾏ 的数据

常用方法：

  • getObject(String str) //获取任意类型的数据

  • getString(String str) //获取指定类型的数据

//对结果集进行滚动查看的方法

  • next() //游标向下移动一行

  • Previous()

  • absolute(int row)

  • beforeFirst()

  • afterLast()

5.PreparedStatement对象

PreparedStatement对象继承Statement对象

作用：

  • Statement对象编译SQL语句时，如果SQL语句有变量，就需要使⽤分隔符来隔开，如果变量⾮常

    多，就会使SQL变得⾮常复杂，PreparedStatement可以使⽤占位符，简化sql的编写

  • Statement会频繁编译SQL。PreparedStatement可对SQL进⾏预编译，提⾼效率，预编译的
    SQL存储在PreparedStatement对象中

  • PreparedStatement防⽌SQL注⼊。Statement通过分隔符’++’,编写永等式，可以不需要密码
    就进⼊数据库

  try {
        Class.forName("com.mysql.jdbc.Driver");
        Connection con=DriverManager.getConnection("jdbc:mysql://localhost:3306/student?characterEncoding=utf-8","root","123");
        String sql="insert into user(uid,uname,upwd)values(?,?,?)";
        PreparedStatement ps=con.prepareStatement(sql);
        ps.setString(1,"1");
    } catch (ClassNotFoundException e) {
        e.printStackTrace();
    } catch (SQLException e) {
        e.printStackTrace();
    }
}

解析:

  • 给其占位符来进行赋值 setInt(index,data) setString(index,data)
    第一个参数表示设置索引 索引是从1开始
    第二个参数表示 具体的赋值

Jdbc配置文件读取的封装

import java.io.IOException;
import java.io.InputStream;
import java.util.Properties;
public class JdbcUtils {
    private static JdbcUtils jdbcUtils;//私有的属性，私有的构造
    private JdbcUtils(){              //保证外部类不能实例化这个对象
    }
    public static synchronized JdbcUtils getJdbcUtils(){  //定义方法来实例化这个类
        if(jdbcUtils==null){
            jdbcUtils=new JdbcUtils();
        }
        return jdbcUtils;
    }
    public static Properties properties;
    static{
        try {
            properties=new Properties();
            InputStream fis=JdbcUtils.class.getResourceAsStream("database.properties");//通过类加载器来记载资源文件
            properties.load(fis);  //当前文件 的内容已经加载到properties对象，调用properties对象就可以获得其内容
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    public String getValues(String key){  //通过键值来获取，类似于hashMap对象
        return properties.getProperty(key);
    }
}

import com.mysql.jdbc.ResultSet;
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.SQLException;
public class BaseDao {
    public  Connection getConnection() {  //此方法用于连接对象
        Connection conn = null;
        try {
            Class.forName(JdbcUtils.getJdbcUtils().getValues("jdbc.driver"));
            conn = DriverManager.getConnection(JdbcUtils.getJdbcUtils().getValues("jdbc.url"),
                    JdbcUtils.getJdbcUtils().getValues("jdbc.username"),
                    JdbcUtils.getJdbcUtils().getValues("jdbc.password"));
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        } catch (SQLException e) {
            e.printStackTrace();
        }
        return conn;
    }
    public int update(String sql, Object[] objects) {
        int num = -1;
        Connection con = null;
        PreparedStatement ps = null;
        try {
            con = getConnection();//获取连接对象
            ps = con.prepareStatement(sql);//执行得到的SQL对象
            if (objects != null && objects.length > 0) {//非空验证，判断是否需要拼接占位符
                for (int i = 0; i < objects.length; i++) {
                    ps.setObject((i + 1), objects[i]);//给占位符一个个对应赋值
                }
            }
            num = ps.executeUpdate();
        } catch (SQLException e) {
            e.printStackTrace();
        } finally {
            close(null, ps, con);
        }
        return num;
    }
    public ResultSet selectAll(String sql,Object[] objects){
        ResultSet rs= null;
        try {
            Connection con = getConnection();
            PreparedStatement  ps = con.prepareStatement(sql);
            if (objects != null && objects.length > 0) {
                for (int i = 0; i < objects.length; i++) {
                    ps.setObject(i + 1, objects[i]);
                }
            }
            ps.executeQuery();

        }catch (SQLException e) {
            e.printStackTrace();
        }finally {
        }
        return rs;
    }
    public void close(ResultSet rs,PreparedStatement ps,Connection conn){
        try {
            if(rs!=null){
                rs.close();
            }
            if(ps!=null){
                ps.close();
            }
            if(conn!=null){
                conn.close();
            }
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
